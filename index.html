Welcome to GitHub Pages.

Jazzina Animation Library is a CSS Library or a .css file that contain CSS code of different animations. These animation can used in your HTML file. It is a simple, fast and small size animation library. You just have to put required animation class in your code and have to link jazzina.css in your head section. <head> <title>Jazzina Animation Library - JAL Sample</title> <link rel="stylesheet" href="jazzina.css"> </head>
